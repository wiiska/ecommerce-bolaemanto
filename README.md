# ⚽ Bola e Manto - E-commerce de Camisetas de Futebol

![Versão](https://img.shields.io/badge/versão-1.0.0-blue)
![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

O **Bola e Manto** é um protótipo de e-commerce moderno e responsivo focado na venda de camisetas de futebol (oficiais, personalizadas e retrô). O projeto foi desenvolvido com foco em performance, sem o uso de frameworks externos, utilizando apenas HTML5 e CSS3 puro.

## 🚀 Demonstração

O site apresenta uma interface limpa e intuitiva, com as seguintes seções:
- **Header:** Menu de navegação responsivo com suporte a dispositivos móveis.
- **Hero:** Destaque principal com chamadas para ação (CTA).
- **Categorias:** Filtros visuais para Brasil, Mundo, Seleções e Modelos Retrô.
- **Grid de Produtos:** Vitrine moderna para exibição dos produtos em destaque.
- **Footer:** Rodapé completo com formulário de newsletter e redes sociais.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização avançada utilizando:
  - **Flexbox & CSS Grid:** Para layouts dinâmicos e responsivos.
  - **CSS Variables:** Para fácil manutenção de cores e tipografia.
  - **Arquitetura Modular:** Separação de estilos por componentes.
- **SVG:** Ícones e logo em formato vetorial para maior nitidez.

## 📁 Estrutura de Pastas

```text
ecommerce-bolaemanto/
├── css/
│   ├── components/       # Estilos específicos de cada seção (header, footer, etc.)
│   ├── base.css          # Estilos globais e tipografia
│   ├── reset.css         # Reset de estilos padrão do navegador
│   └── variables.css     # Definição de cores e variáveis globais
├── images/
│   ├── banners/          # Imagens de fundo e hero
│   ├── icons/            # Ícones de interface e redes sociais
│   ├── logo/             # Logo oficial da marca
│   └── products/         # Fotos dos produtos da loja
├── index.html            # Página principal do projeto
└── README.md             # Documentação do projeto
```

## 💻 Como Executar o Projeto

1. Clone o repositório ou baixe os arquivos.
2. Certifique-se de que a estrutura de pastas `css/` e `images/` esteja no mesmo nível do arquivo `index.html`.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
4. (Opcional) Utilize a extensão **Live Server** no VS Code para visualizar as alterações em tempo real.

## 📱 Responsividade

O projeto foi construído pensando em dispositivos móveis primeiro (Mobile-First), garantindo que a experiência de compra seja excelente tanto em smartphones quanto em desktops.
